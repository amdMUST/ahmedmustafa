Playboi Carti TwitterBot

X

@cartibotslatt

[Sep 2019 - Oct 2019]

For fun, I made a twitter bot that will tweet lyrics from the rapper, Playboi Carti, songs. It will also respond if anyone @'s him with a response from a text file. It will also wait for Playboi Carti to tweet and reply as soon as he tweets, and be able to respond to anyone tweeting with the keyword 'carti.' The last two features aren't currently deployed because of server costs, but it works, and you can look in the GitHub files to see the code. I learned Python and how to use Twitter's API with tweepy. This project taught me how to read the API documentation and implement it with my code. Overall a great project and I hope to add some more features when I have some free time.

Check it out on GitHub! Check out on twitter! @cartibotslatt
